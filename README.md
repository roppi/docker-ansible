# docker-ansible

## ディレクトリ構成

```
.
├── README.md
├── ansible
│   ├── hosts
│   └── sample-playbook.yml
├── docker
│   ├── ansible
│   │   └── Dockerfile
│   └── target
│       └── Dockerfile
└── docker-compose.yml
```

## 使い方

1. Dockerコンテナを起動します。

```
$ docker compose up -d
```

2. Ansibleコンテナに接続します。

```
$ docker exec -it ansible-controller /bin/ash
```

3. Playbookを実行します

```
# ansible-playbook -i hosts sample-playbook.yml
```

4. 以下ように表示されれば、成功です。

